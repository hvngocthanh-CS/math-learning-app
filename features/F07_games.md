# F07 - Game System

**Phase**: 3
**Screen**: S-06 (Game Lobby)

## Mo ta

Kho tro choi toan hoc giup hoc sinh luyen tap qua cac mini-game thu vi.

## Chuc nang chi tiet

### Game Lobby (S-06)
- Danh sach game dang Card-based UI
- Moi game card hien thi:
  - Ten game
  - Hinh anh minh hoa
  - **Reward**: So xu nhan duoc
  - **Difficulty**: Do kho (De / Trung binh / Kho)
  - Trang thai: Moi / Da choi / High score

### Cac loai mini-game (de xuat)
- **Tinh nhanh**: Tra loi phep tinh trong thoi gian gioi han
- **Ghep doi**: Ghep phep tinh voi ket qua dung
- **Dua xe so**: Tra loi dung de tang toc xe
- **Xay nha**: Giai toan de xay cac tang nha

### Co che chung
- Thoi gian gioi han (countdown)
- Tinh diem theo do chinh xac + toc do
- Reward xu khi hoan thanh
- Luu high score
