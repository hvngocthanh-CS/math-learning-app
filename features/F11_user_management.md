# F11 - User Management

**Phase**: 4
**Screen**: S-12

## Mo ta

Quan ly nguoi dung va phan quyen trong he thong.

## Chuc nang chi tiet

### Danh sach user
- Hien thi tat ca nguoi dung trong he thong
- Filter theo role (Student / Parent / Admin)
- Search theo ten, email, SDT
- Pagination

### Phan quyen
- Gan role cho nguoi dung
- Teacher/Admin tao tai khoan cho Student va Parent
- Thay doi quyen truy cap

### Audit Logs
- Track tat ca hoat dong quan trong:
  - Thay doi diem so
  - Cap nhat noi dung bai hoc
  - Hoat dong user (dang nhap, thay doi thong tin)
  - Tao/xoa tai khoan
- Hien thi theo timeline
- Filter theo loai hoat dong, user, thoi gian
