# F06 - Gamification & Reward System

**Phase**: 2
**Screen**: S-08 (My Profile & Shop)

## Mo ta

He thong diem thuong, huy hieu, shop va profile ca nhan de tang dong luc hoc tap.

## Chuc nang chi tiet

### He thong diem thuong
- **Sao**: Nhan khi hoan thanh bai hoc/quiz (1-3 sao)
- **Xu (Coins)**: Tien te trong app, dung de mua item
- **EXP**: Kinh nghiem, tang level
- **Huy hieu (Badges)**: Dat duoc khi hoan thanh muc tieu dac biet

### Cach nhan thuong
- Hoan thanh bai hoc -> Sao + Xu
- Streak (chuoi ngay hoc) -> Bonus xu
- Choi game -> Xu
- Dat thanh tich -> Huy hieu

### Profile (S-08)
- Avatar (co the thay doi)
- Badge collection (bo suu tap huy hieu)
- Level hien tai + EXP bar
- Thong ke: Tong sao, tong xu, so bai da hoan thanh

### Shop (S-08)
- Mua Skins (giao dien)
- Mua Items (vat pham trang tri)
- Thanh toan bang xu
- Hien thi gia va trang thai (da mua / chua mua)

## Reward Loop
```
Hoc bai -> Sao + Xu -> Mua item o Shop -> Trang tri Profile
    |                                           |
    +--- Streak -> Bonus xu ---+               |
    +--- Game -> Xu -----------+               |
                                -> Tang dong luc hoc tiep
```
